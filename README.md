# Pokémon Team Builder

A web application that generates a balanced Pokémon team based on a Pokémon selected by the user.

## Features

- Search any Pokémon
- Team generation
- Pokémon API integration
- Responsive design
- Team analysis

## Technologies

- HTML
- CSS
- JavaScript
- PokeAPI

## What I learned

- Fetch API
- Asynchronous JavaScript
- DOM manipulation
- Responsive layouts
- Working with external APIs
